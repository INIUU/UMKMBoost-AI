📘 Penjelasan Aplikasi – FinSight
FinSight adalah aplikasi manajemen keuangan sederhana namun cerdas, dirancang untuk membantu pelaku usaha kecil, UMKM, atau individu mencatat modal, pendapatan, dan menghitung estimasi waktu balik modal (BEP – Break Even Point).

Aplikasi ini memadukan pencatatan finansial dasar dengan AI gratis sebagai nilai tambah agar aplikasi terlihat modern dan helpful.

🎯 Apa Tujuan FinSight?
FinSight dibuat untuk memudahkan pengguna yang sering bingung dalam:

Mencatat modal awal

Menghitung total pengeluaran operasional

Menghitung keuntungan harian/bulanan

Mengetahui kapan modal bisa kembali (BEP)

Mendapat insight otomatis tanpa harus belajar akuntansi

Dengan FinSight, semua itu bisa dihitung otomatis.

✨ Fitur Utama FinSight
1. Dashboard Keuangan
Menampilkan:

Total Modal

Total Pengeluaran

Total Pendapatan

Total Keuntungan Bersih

Estimasi Waktu Balik Modal

Semua dihitung otomatis berdasarkan data yang diinput pengguna.

2. Input Modal dan Pengeluaran
Pengguna dapat memasukkan:

Modal awal

Pengeluaran rutin (misal: listrik, bahan baku, gaji)

Pengeluaran tidak rutin

Aplikasi akan otomatis mengelompokkan dan menganalisis data.

3. AI Financial Insight (AI Gratis)
Menggunakan model open-source (seperti Claude Code AI / HuggingFace atau lainnya), AI akan memberikan insight seperti:

“Pengeluaran bulan ini naik 12% dibanding bulan lalu.”

“Jika pemasukan stabil, modal akan kembali dalam 27 hari.”

“Kategori pengeluaran terbesar: Bahan baku.”

AI hanya menganalisis teks dan angka, jadi tidak membutuhkan API image yang mahal, tetap gratis.

4. Break Even Point Calculator
FinSight otomatis menghitung:

Keuntungan per hari/bulan

Total modal

Estimasi kapan modal balik (BEP)

Contoh:

“Dengan keuntungan rata-rata Rp 80.000 per hari, modal Rp 2.000.000 akan balik dalam 25 hari.”

5. Laporan Otomatis
Pengguna bisa melihat:

Laporan harian

Laporan bulanan

Tren grafik sederhana

🧠 Teknologi yang Digunakan
Laravel – backend & API

Model open-source lokal (contoh: Llama 3, Mistral, Gemma)
